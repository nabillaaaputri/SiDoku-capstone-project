## рҹ“Ў Data Flow

### User Registration & Login Flow
```
User Input вҶ’ Frontend Form вҶ’ Backend Auth API вҶ’ Database
                                    вҶ“
                            Validate & Hash Password
                                    вҶ“
                            Create User Record
                                    вҶ“
                            Generate JWT Token
                                    вҶ“
                            Return Token to Frontend
                                    вҶ“
                            Store Token (localStorage)
```

### Transaction Recording Flow
```
User Input вҶ’ Frontend Form вҶ’ Backend API вҶ’ AI Service (Classify) вҶ’ Database
                                             вҶ“
                                    Categorize Transaction
                                             вҶ“
                                    Return Category
                                    вҶ“
                            Save to Database
                                    вҶ“
                            Update Dashboard Cache
                                    вҶ“
                            Return Confirmation
```

### Dashboard Update Flow
```
Frontend Load вҶ’ Request Dashboard Data вҶ’ Backend API
                                            вҶ“
                                    Query Database
                                    (Transactions, Products, etc)
                                            вҶ“
                                    Aggregate Data
                                            вҶ“
                                    Return to Frontend
                                            вҶ“
                                    Render Charts & Cards
                                    (using Recharts)
```

## рҹ“Ҡ Technology Decision Rationale

| Layer | Tech | Why? |
|-------|------|------|
| Frontend | React.js | Component reusability, large ecosystem |
| Styling | Tailwind CSS | Fast development, utility-first approach |
| Backend | Express.js | Lightweight, perfect for REST API |
| Database | PostgreSQL | Relational data, robust, free tier support |
| AI/ML | Python + FastAPI | Best ML libraries (TensorFlow, scikit-learn) |
| Hosting | Vercel/Railway | Free tier, easy CI/CD, good for learning |

## рҹ”„ Integration Points

1. **Frontend вҶ” Backend**: REST API with Axios
2. **Backend вҶ” AI Service**: HTTP requests to FastAPI
3. **Backend вҶ” Database**: SQL queries via Node.js drivers
4. **Frontend вҶ” Local Storage**: Auth tokens, user preferences
5. **CI/CD**: GitHub Actions вҶ’ Vercel/Railway deployments
