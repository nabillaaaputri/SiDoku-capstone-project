# SiDoku Frontend
SiDoku (Sistem Data Operasional dan Keuangan Usaha) adalah aplikasi berbasis web yang membantu pelaku UMKM dalam mencatat dan mengelola data operasional usaha, keuangan, stok barang, serta memperoleh insight bisnis melalui teknologi kecerdasan buatan (AI).

## Demo Aplikasi
https://sidoku.vercel.app/

## Fitur Utama
- Dashboard Ringkasan Usaha
- Manajemen Data Barang
- Pencatatan Stok Masuk
- Pencatatan Stok Keluar
- Pencatatan Pengeluaran
- Rekap Penjualan
- Insight dan Analisis Bisnis Berbasis AI
- Prediksi Penjualan
- Chatbot Asisten AI
- Login dan Registrasi Pengguna

## Teknologi yang Digunakan
- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Axios
